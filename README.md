# Anthony Monaghan

Source files for [anthonymonaghan.com](https://anthonymonaghan.com/), the personal website of Anthony Monaghan, author of *Snug*.

The site is intentionally simple and static. It is built with HTML and CSS and deployed through GitHub Pages.

## Structure

- `index.html` – page content and styling
- `tony-headshot.jpg` – portrait used on the site
- `social-preview.jpg` – social sharing image
- `favicon.*`, `apple-touch-icon.png`, `icon-*.png` – browser and device icons
- `site.webmanifest` – web app metadata
- `robots.txt` and `sitemap.xml` – search engine metadata
- `CNAME` – custom domain configuration

## Deployment

Changes pushed to the `main` branch are published automatically through GitHub Pages.
